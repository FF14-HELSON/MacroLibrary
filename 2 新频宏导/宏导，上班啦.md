### 如何获得坐骑    
```
/micon 随机坐骑
/b 《如何获得坐骑》
/b 【条件】 完成20级主线任务《前行之路》，以及20级蓝色重要任务《我的专属陆行鸟》
          玩家在《前行之路》选择加入的大国防联军不同，则《我的专属陆行鸟》任务起始点不同
          黑涡团 任务起始点 利姆萨·罗敏萨上层甲板(13, 13)
          双蛇党 任务起始点 格里达尼亚新街(10, 11)
          恒辉队 任务起始点 乌尔达哈现世回廊(8, 9)
/b 【说明】 任务完成后，需要《背包 -> 专属陆行鸟召唤笛 -> 使用》操作后，陆行鸟才加入坐骑列表；
/b         此后，使用《角色 -> 坐骑一览 -> 随机坐骑/召唤指定坐骑》骑乘坐骑。
/b         建议将图标设置到热键栏方便使用。
```

### 什么是金蝶游乐场    
```
/micon 金蝶币
/b 《如何开启金蝶游乐场》
/b 【条件】 完成15级蓝色重要任务《前往游乐场》，起始点 乌尔达哈现世回廊(9, 9)    
/b 【说明】 与金蝶游乐城的水晶共鸣后，可以在《传送 -> 乌尔达哈》下找到对应水晶；
/b       忘记共鸣时，可以从《乌尔达哈飞艇坪》乘坐飞艇前往金蝶游乐场。
```
